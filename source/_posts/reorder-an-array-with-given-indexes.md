## [We have an array of objects A and an array of indexes B. Reorder objects in array A with given indexes in array B. Do not change array A's length.](https://www.careercup.com/question?id=5756151524229120)

#### example:

```javascript
var A = [C, D, E, F, G];
var B = [3, 0, 4, 1, 2];
sort(A, B);
// A is now [D, F, G, C, E];
```
