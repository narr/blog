## Quick Start

### Clone

``` bash
$ git clone https://github.com/narr/blog.git
```

### Install

``` bash
$ npm install
```

### Run development server

``` bash
$ npm run server
```

Open a browser on localhost:8080

### Deploy to [Github](https://github.com/narr/blog/tree/gh-pages)

``` bash
$ npm run deploy
```
